# Turnero Gym

## Descripción

Turnero Gym es una aplicación web interactiva desarrollada como proyecto final del curso de JavaScript. Su objetivo es simular el proceso de reserva de actividades dentro de un gimnasio, permitiendo a los usuarios visualizar los turnos disponibles, consultar la cantidad de cupos y realizar reservas de manera dinámica.

La aplicación utiliza un archivo JSON para almacenar los datos de las actividades y la API Fetch para cargarlos, simulando el consumo de información desde una fuente externa.

---

## Funcionalidades

* Visualización de actividades disponibles.
* Consulta de cupos disponibles por actividad.
* Reserva de turnos en tiempo real.
* Actualización automática de cupos.
* Bloqueo de reservas cuando no existen cupos disponibles.
* Notificaciones visuales mediante SweetAlert2.
* Interfaz dinámica generada con JavaScript y manipulación del DOM.
